# Worklog

## [Sprint 1-3] Core Development & Stability (V0.1 ~ V1.01)

### 1. Functional Baseline
- **Hypothesis**: 크로스핏 운동 환경에 최적화된 고정밀 모바일 타이머 구축.
- **Results**: 
  - 4개 핵심 운동 모드 구현 (AMRAP, FOR TIME, TABATA, EMOM).
  - Date.now() 기반 100% 정확도 달성.
  - 시작 전 5초 준비 시간 및 저음 버저 사운드 추가.

### 2. High-End Mobile Features (V1.01)
- **Features**: 
  - **Wake Lock API**: 운동 중 화면 꺼짐 방지.
  - **Full-Screen API**: 운동 중 몰입감 극대화.
  - **Tap Debouncing**: 1.5초 터치 잠금으로 오작동 방지.
  - **Precision Logging**: 랩 타임 및 갭 기록 (소수점 1자리).
  - **Tabata Customization**: 인터벌 시간 및 라운드 조절 기능.
